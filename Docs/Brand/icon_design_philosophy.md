# Proxy Mesh

Proxy Mesh treats routing as calm infrastructure rather than spectacle. The symbol should feel like a control point that quietly coordinates many upstream paths with precision. Form is reduced to a central proxy hub, disciplined links, and several endpoint nodes. The work must read instantly at menu bar scale while still feeling meticulously constructed at app-icon scale.

Color stays flat and deliberate. The primary field uses `#339966` to communicate calm routing, trust, and stable traffic flow, while a brighter cyan accent marks active forwarding. Contrast is graphic and clean, with no gloss, texture, or ornamental gradients competing with legibility.

Composition is radially stable. The proxy hub anchors the center while outer nodes orbit with measured spacing so the symbol communicates "one proxy, multiple connections" without clutter. Every curve and line should look mathematically placed and patiently refined.

Scale changes preserve the same idea. At large size, the icon shows a fuller network motif; at tray size, the language collapses into a distilled glyph with a central core and a few clear nodes. The family should feel precise, quiet, and infrastructure-first.
